常用标签和标签语义

学习标签有技巧，重点记住标签语义

  1. 标题标签

    “

    -

    (分别是一级标题到六级标题)”

    —

    一级标题

    —

    二级标题

    —

    三级标题

    —

    四级标题

    —

    五级标题

    —

    六级标题

  2. 段落标签和换行标签，段落标签（文字分段）换行标签（强制换行）

    在网页中，要把文字分段显示，需要用到段落标签

    和

  3. 文本格式化标签

    在网页中，有时需要为文字设置粗体，斜体，下划线效果，就需要用到文本格式化标签，让文字特殊显示

    语义 标签 说明
    加粗 strong>或者> 推荐使用标签加粗，语义更强烈
    倾斜 或者 推荐使用标签加粗，语义更强烈
    删除线 或者 推荐使用标签加粗，语义更强烈
    下划线 或者 推荐使用标签加粗，语义更强烈

  4. 布局标签

    “

    和”标签，无语义标签，常用布局，相当于一个盒子容器，“
    ”独占一行，“”也用来布局，一行可以放多个

  5. 图像标签和路径

    ““

    属性 属性值 说明
    src 图片路径 必须属性
    alt 文本 替换文本
    title 文本 提示文本
    width 像素 图像宽度
    height 像素 图像高度
    border 像素 图像边框粗细

  6. 超链接标签

    在HTML中标签用于定义超链接，作用是从一个页面跳转到另一个页面。

    文本或者图像

    分类：

    - 外部链接
- 内部链接
- 下载链接
- 网页元素链接
- 空链接
- 锚点链接（点击标签跳转到特定位置）

  7. 注释和特殊字符

    使用快捷键-> Ctrl + / 快速注释

    7.1 特殊字符

  1. 表格标签

    表格不是用来布局页面的，而是用来展示数据的

    1. “
      ”用于定义表格的标签
    2. ““用于定义表格中的行，必须放在table标签内
    3. “用于定义表格中的单元格，放在tr内”
    4. “”用于定义表头
    5. “”表格的头部区域
    6. “”表格的主体部分

  2. 列表标签

    无序列表ul / 有序列表 ol

    • 无序列表没有顺序之分，是并列关系
    • ul/ol/dl中只能嵌套li,不能在ul中直接嵌套其他标签或者文字
    • li相当于一个容器，可以容纳所有元素
    • 无序列表带有自己的属性，实际使用会将其样式去掉

  3. 自定义列表dl

  4. 表单标签

    由三部分组成，分别是表单域，表单控件，提示信息，通过form标签声明一个表单域

- input输入表单元素

  | 属性值   | 描述                                                   |
  | -------- | ------------------------------------------------------ |
  | button   | 定义按钮                                               |
  | checkbox | 定义复选框                                             |
  | file     | 定义输入字段和浏览按钮，供文件上传                     |
  | hidden   | 定义隐藏的输入字段                                     |
  | image    | 定义图像形式的提交按钮                                 |
  | password | 定义密码字段                                           |
  | radio    | 定义单选按钮                                           |
  | reset    | 定义充值按钮，会清除表单中的所有数据                   |
  | submit   | 定义提交按钮，将表单数据提交到服务器                   |
  | text     | 定义单行输入字段，用户可在其中输入文本，默认为20个字符 |

  | 属性      | 属性值     | 描述               |
  | --------- | ---------- | ------------------ |
  | name      | 用户自定义 | input元素的名称    |
  | value     | 用户自定义 | input元素的值      |
  | checked   | checked    | 选中               |
  | maxlength | 正整数     | 输入字符的最大长度 |

  * label标签 为input定义标注

    用于绑定一个表单元素，点击label标签内的文本，浏览器就会自动将光标转到对应的表单元素上，增加用户体验,for属性的值应该与被绑定标签的id值一致
